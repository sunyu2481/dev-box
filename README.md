# dev-box

A multi-architecture development container image built with GitHub Actions and published to GitHub Container Registry (GHCR).

## Features

- Ubuntu 24.04 base
- Node.js
- pnpm
- Python
- uv
- Go
- Java 21
- Maven
- Gradle
- Git / Git LFS
- Common development tools
- AI CLI tools:
  - Claude Code
  - OpenAI Codex CLI
  - Gemini CLI

## Image

```bash
docker pull ghcr.io/sunyu2481/dev-box:latest
